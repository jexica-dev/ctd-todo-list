# ctd-todo-list

A simple React todo list app for tracking and displaying tasks.

## Installation

Make sure you have [Node.js](https://nodejs.org/) installed, then clone the repo and install dependencies:

```bash
git clone https://github.com/jexica-dev/ctd-todo-list.git
cd ctd-todo-list
npm install
```

## Running the Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173` (if using Vite) or `http://localhost:3000` (if using Create React App).
